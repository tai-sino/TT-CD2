    Tao SinhVienController.php voi 5 methods:

    1. `public function import(Request $request)`:
       - Validate: `['file' => 'required|file|mimes:xlsx,xls', 'ky_lvtn_id' => 'required|exists:ky_lvtn,id']`
       - Dung `PhpOffice\PhpSpreadsheet\IOFactory::load($request->file('file')->getPathname())`
       - Goi `$spreadsheet->getActiveSheet()->toArray()`
       - Skip row dau tien (header)
       - Loop rows: trim tung cell, validate MSSV va hoTen khong rong, check MSSV trung trong DB
       - Neu loi: push vao `$errors[] = ['row' => $i + 1, 'message' => '...']`, continue
       - Neu ok: `SinhVien::create([...])` voi `matKhau => Hash::make($mssv)` va `ky_lvtn_id => $request->ky_lvtn_id`
       - QUAN TRONG (KY-03): Moi SV duoc import phai co ky_lvtn_id — lay tu request, KHONG de null
       - Return: `response()->json(['imported' => $imported, 'errors' => $errors])`
       - KHONG set Content-Type header tren response — de Laravel tu set

    2. `public function index(Request $request)`:
       - Query SinhVien, ho tro filter:
         - `ky_id`: `where('ky_lvtn_id', $request->ky_id)` khi co
         - `lop`: `where('lop', $request->lop)` khi co
         - `gvhd_id`: filter SV theo GVHD da phan cong — `whereHas('deTai', fn($q) => $q->where('maGV_HD', $request->gvhd_id))` khi co (SV-03: filter theo GVHD)
         - `search`: `where(function($q) { $q->where('mssv','like',"%$s%")->orWhere('hoTen','like',"%$s%"); })` khi co
       - Paginate 20 per page: `->paginate(20)`
       - Return: `response()->json($query)` (Laravel paginate tu co data, total, per_page, current_page)

    3. `public function store(Request $request)`:
       - Validate: `['mssv' => 'required|unique:sinhvien,mssv', 'hoTen' => 'required', 'email' => 'nullable|email|unique:sinhvien,email', 'lop' => 'nullable', 'ky_lvtn_id' => 'required|exists:ky_lvtn,id']`
       - Create voi matKhau = Hash::make($request->mssv)
       - Return 201

    4. `public function update(Request $request, $mssv)`:
       - Tim SinhVien by mssv, 404 neu khong co
       - Validate: `['hoTen' => 'required', 'email' => 'nullable|email|unique:sinhvien,email,'.$mssv.',mssv', 'lop' => 'nullable', 'ky_lvtn_id' => 'nullable|exists:ky_lvtn,id']`
       - QUAN TRONG (KY-03): ky_lvtn_id phai nam trong validation rules de admin co the chuyen SV sang ky khac
       - Update hoTen, lop, email, ky_lvtn_id (khong doi mssv)
       - Return 200

    5. `public function destroy($mssv)`:
       - Tim SinhVien by mssv, 404 neu khong co
       - Check neu SV co maDeTai (da gan de tai) -> tra 422 voi message "Khong the xoa sinh vien da co de tai"
       - Delete va return `['message' => 'ok']`

| Threat ID | Category | Component | Disposition | Mitigation |
|-----------|----------|-----------|-------------|------------|
| T-02-01 | Tampering | SinhVienController::import | mitigate | Validate file mimes:xlsx,xls; phpspreadsheet chi doc data, khong execute macro |
